# HangGlider
山间滑行
使用WorldMachine生成的雪地地图，maps文件超过GitHub 50MB限制，我由于没有购买WorldMachine的商业授权，所以不提供地图模型，但场景其他物品齐全，可自行搭建。
引擎版本4.23.0-preview-6
